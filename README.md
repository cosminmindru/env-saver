## Usage

```sh-session
$ npm install -g env-saver
$ env-saver COMMAND
running command...
$ env-saver (-v|--version|version)
env-saver/0.0.0 linux-x64 node-v14.16.0
$ env-saver --help [COMMAND]
USAGE
  $ env-saver COMMAND
...
```
